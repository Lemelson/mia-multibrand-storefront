# UX/UI аудит витрины MIA — 2026-06-09

Базовый разбор перед серией улучшений. Осматривалось вживую (десктоп 1440px, мобайл 375px) + сверка с исходниками.

Легенда: 🔴 баг/недоделка · 🟡 UX-проблема · 🟢 идея/полировка.

## Дизайн-система
- 🔴 Задуман serif-дисплей (`Cormorant Garamond`) + `Manrope`, но в `layout.tsx` жёстко стоит `data-font-theme="montserrat"` и грузится только Montserrat → нет типографического контраста, лого/заголовки/текст одним гротеском. Плюс Cormorant Garamond не поддерживает кириллицу — для русскоязычной витрины нужен кириллический serif.
- 🟢 Палитра тёплая, нейтральная — уместна. Сетка `max-w-content: 1440px`.

## Шапка
- 🟡 Дублирование: бургер + инлайн-меню «Женщинам/Мужчинам/Детям».
- 🟡 Меню открывается по hover на бургер (случайные открытия).
- 🔴 Выбор магазина `hidden md:block` — недоступен на мобильном, хотя влияет на наличие.

## Меню (выезжающая панель)
- 🟡 Плоская иерархия, узкая панель на десктопе.
- 🔴 `.slice(0, 8)` обрезает женские категории (нет «Сумки»/«Аксессуары»).
- 🟡 «Детям → Детское» тавтология; «Бренды» ведёт на `/catalog?brands=` (пусто).

## Главная — Hero
- 🔴 Текст `absolute left-0` игнорирует отступы контейнера → обрезается на мобильном.
- 🟡 Слабый оверлей `bg-black/20` — белый текст не читается на светлых слайдах.
- 🟡 Автокарусель без точек/стрелок/паузы; CTA-ссылка меняется каждые 5 c.
- 🔴 Тяжёлые внешние Unsplash-изображения в hero и «О магазине».

## Главная — секции / футер
- 🟡 «Наши бренды» — некликабельные текстовые чипы.
- 🔴 Публичная ссылка «Панель управления» → `/admin` в футере.
- 🔴 Переключатель шрифтов (`FooterFontSwitcher`) виден всем пользователям (демо-инструмент).

## Каталог / фильтры
- 🟡 Нет счётчика результатов; нет «Сбросить всё» / чипов активных фильтров.
- 🟡 Непоследовательность: «Бренд» одиночный и без отражения выбора на кнопке; размер/цвет мультивыбор.
- 🟡 Только 3 колонки на широких экранах (`xl:grid-cols-3`).
- 🔴 Мобайл: список из 11 категорий занимает весь первый экран; панель фильтров переносится на 3 строки, мелкие тап-таргеты.

## Карточка товара (PDP)
- 🔴 Не показывается скидка (только `product.price`, без старой цены/процента).
- 🔴 Кнопка «♡ В избранное» без обработчика — нерабочая.
- 🟡 Авто-сгенерированные названия/описания («Смесовая шерсть Свитер Twinset»).
- 🟡 Related-сетка 4 колонки vs 3 в каталоге.
